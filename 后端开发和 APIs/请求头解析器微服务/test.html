Instructions

To run this microservice locally:

  1. Make sure you have Node.js installed
  2. Open a terminal/command prompt
  3. Navigate to this project directory
  4. Run: npm install
  5. Run: npm start
  6. The server will start on http://localhost:3000
  7. Test the API endpoint: http://localhost:3000/api/whoami
