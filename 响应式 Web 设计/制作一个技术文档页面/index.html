介绍

JavaScript 是一种高级的、解释型的编程语言，主要用于网页开发。

它最初由 Brendan Eich 在 1995 年为 Netscape Navigator 浏览器创建。

JavaScript 是一种动态类型语言，支持面向对象、命令式和函数式编程范式。

在现代 Web 开发中，JavaScript 已经成为不可或缺的技术。

它可以在浏览器中运行，也可以在服务器端运行（Node.js）。

console.log("Hello, World!"); let message = "Welcome to JavaScript";
  • 客户端脚本语言
  • 支持异步编程
  • 丰富的生态系统

变量和数据类型

JavaScript 中的变量用于存储数据值。

使用 let、const 或 var 关键字声明变量。

JavaScript 有七种基本数据类型：

  • Number（数字）
  • String（字符串）
  • Boolean（布尔值）
  • Undefined（未定义）
  • Null（空值）
  • Symbol（符号）
  • Object（对象）
let name = "张三"; const age = 25; let isStudent = true;

函数和对象

函数是 JavaScript 中的一等公民，可以像变量一样传递和使用。

对象是键值对的集合，用于表示复杂的数据结构。

函数可以通过多种方式定义：函数声明、函数表达式、箭头函数等。

对象可以通过字面量语法、构造函数或 Object.create() 方法创建。

function greet(name) { return `Hello, ${name}!`; } const person = { name: "李四", age: 30 };
  • 函数声明
  • 函数表达式
  • 箭头函数
  • 对象字面量
  • 构造函数

数组和循环

数组是 JavaScript 中用于存储多个值的有序集合。

数组可以包含不同类型的元素，并且长度可以动态变化。

JavaScript 提供了多种循环方式来遍历数组和对象。

常用的循环方法包括 for、while、forEach、map、filter 等。

const numbers = [1, 2, 3, 4, 5]; for (let i = 0; i < numbers.length; i++) { console.log(numbers[i]); }
  • for 循环
  • while 循环
  • forEach 方法
  • map 方法
  • filter 方法

DOM操作

DOM（文档对象模型）是 HTML 文档的编程接口。

JavaScript 可以通过 DOM API 来操作网页的内容、结构和样式。

常见的 DOM 操作包括：选择元素、修改内容、添加/删除元素、事件处理等。

现代 JavaScript 提供了强大的 DOM 操作方法，使网页交互更加丰富。

document.getElementById("myElement"); element.addEventListener("click", function() { alert("Clicked!"); });
  • 元素选择
  • 内容修改
  • 事件处理
  • 样式操作
  • 元素创建和删除
